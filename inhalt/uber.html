Über das Schoolallroundsystem

Jugendhackt Hamburg

Das Schoolallroundsystem entsteht in Zusammenarbeit mit Jugenhackt Hamburg 2017. Wir arbeiteten vom 9.6. bis zum 11.6. und starteten von nichts. Durch Hilfe der Mentoren/innen wurde dieses Projekt wirkllich groß.

Open Source

Der Code dieses Projekts ist öffentlich zugänglich und ist unter GitHub erreichbar. So könnt ihr Tipps geben oder den Code selbst Runterladen um ihn für euch zu bearbeiten.
